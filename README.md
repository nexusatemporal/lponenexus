# Nexus Atemporal - Landing Page

Landing page oficial do sistema **One Nexus Atemporal**, um ERP/CRM completo para clínicas de estética.

## Tecnologias

- HTML5
- CSS3 (Design System customizado)
- JavaScript (Vanilla)
- Google Fonts (Raleway, Noto Sans)
- Phosphor Icons

## Estrutura

```
├── index.html      # Página principal
├── style.css       # Estilos
├── script.js       # Interações
└── assets/         # Imagens e logos
```

## Deploy

Faça o clone e sirva os arquivos com qualquer servidor web (Nginx, Apache, etc).
